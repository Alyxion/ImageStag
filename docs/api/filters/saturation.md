# Saturation


![Saturation example](../gallery/filters/saturation.jpg)

Adjust color saturation.

factor: 0.0 = grayscale, 1.0 = original, 2.0 = vivid

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `factor` | float | 1.0 | 0.0 = grayscale, 1.0 = original, 2.0 = vivid |

## Frameworks

Native support: PIL
