# ExtractChannel


![ExtractChannel example](../gallery/filters/extractchannel.jpg)

Extract a single channel from an image.

Outputs a grayscale image with metadata['channel'] set.

:param channel: Channel to extract ('R', 'G', 'B', 'A', or integer index)

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `channel` | str | 'R' | Channel to extract ('R', 'G', 'B', 'A', or integer index) |

## Frameworks

Native support: RAW, CV
