# Sharpness


![Sharpness example](../gallery/filters/sharpness.jpg)

Adjust image sharpness.

factor: 0.0 = blurry, 1.0 = original, 2.0 = sharper

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `factor` | float | 1.0 | 0.0 = blurry, 1.0 = original, 2.0 = sharper |

## Frameworks

Native support: PIL
