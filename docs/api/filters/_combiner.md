# Combiner Filters

## [Blend](./blend.md)

Blend two branches together using a blend mode.

**Parameters:** `mode`, `opacity`

## [Composite](./composite.md)

Composite foreground over background using a mask.

## [FilterGraph](./filtergraph.md)

Directed acyclic graph of filter operations.

**Parameters:** `branches`, `output`, `source`, `nodes`, ...

## [MaskApply](./maskapply.md)

Apply mask to image (set alpha from mask).

## [SizeMatcher](./sizematcher.md)

Match dimensions of two images.

**Parameters:** `mode`, `aspect`, `crop`, `interp`, ...
