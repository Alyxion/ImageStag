# Smooth


![Smooth example](../gallery/filters/smooth.jpg)

Smoothing filter using PIL.

Parameters:
    strength: 'normal' or 'more'

Example:
    'smooth()' or 'smooth(more)'

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `strength` | str | 'normal' | 'normal' or 'more' |

## Examples

```
smooth()
```

## Frameworks

Native support: PIL
