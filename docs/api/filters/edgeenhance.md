# EdgeEnhance


![EdgeEnhance example](../gallery/filters/edgeenhance.jpg)

Enhance edges in image.

Uses PIL's edge enhancement filters.

Parameters:
    strength: 'normal' or 'more'

Example:
    'edgeenhance(normal)' or 'edgeenhance(more)'

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `strength` | str | 'normal' | 'normal' or 'more' |

## Examples

```
edgeenhance(normal)
```

## Frameworks

Native support: PIL
