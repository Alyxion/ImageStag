# Contrast


![Contrast example](../gallery/filters/contrast.jpg)

Adjust image contrast.

factor: 0.0 = gray, 1.0 = original, 2.0 = high contrast

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `factor` | float | 1.0 | 0.0 = gray, 1.0 = original, 2.0 = high contrast |

## Frameworks

Native support: PIL
