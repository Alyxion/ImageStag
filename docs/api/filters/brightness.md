# Brightness


![Brightness example](../gallery/filters/brightness.jpg)

Adjust image brightness.

factor: 0.0 = black, 1.0 = original, 2.0 = 2x bright

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `factor` | float | 1.0 | 0.0 = black, 1.0 = original, 2.0 = 2x bright |

## Frameworks

Native support: PIL
