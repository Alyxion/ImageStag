# API Browser Truncates Session Name

## Description
The API browser UI truncates/shortcuts the session name, making it difficult to see the full session identifier.

## Steps to Reproduce
1. Open the API browser
2. Look at the session name display
3. Observe: session name is truncated/shortened

## Expected Behavior
The full session name/ID should be visible, or at least easily accessible (e.g., via tooltip or expandable field).

## Suggested Fix
- Show full session ID
- Or add tooltip with full ID on hover
- Or add copy button to copy full session ID
