# Sharpen Tool Does Nothing or Creates Weird Pattern

## Description
The Sharpen tool either has no visible effect or creates an unexpected pattern instead of sharpening the image.

## Steps to Reproduce
1. Open an image
2. Select the Sharpen tool
3. Paint over an area
4. Observe: either nothing happens or a strange pattern appears

## Expected Behavior
The painted area should become sharper with enhanced edges and details.

## Affected Files
- `stagforge/frontend/js/tools/SharpenTool.js` (likely)
